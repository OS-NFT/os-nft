# OSNFT – Open Source Non-Fungible Tokens on Cardano

**Bringing large open-source datasets on-chain as NFTs** for immutable provenance, verification, and community access.

## What is OSNFT?
OSNFT tokenizes truly open-source / public-domain datasets (e.g., scientific, government, ML corpora) as Cardano-native NFTs using CIP-25 metadata standards.  
- Data lives on IPFS (with hashes for integrity)  
- Goal: Put impactful data on-chain.

## Why Cardano?
- Low fees, sustainable PoS  
- Native assets & metadata support  
- Strong open-source & decentralization ethos

## Current Status
- MVP phase: Minting process established  
- First datasets in planning (e.g., public congressional records, climate data subsets, ML benchmarks)  
- Infrastructure: Ubuntu, Python, Blockfrost IPFS

## Roadmap
1. Mint 3–5 test NFTs with sample open datasets (Q1 2026)  
2. Public dataset submission form / community voting  
3. Simple frontend explorer for minted OSNFTs
